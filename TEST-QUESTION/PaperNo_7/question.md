# PaperNo_7

## Overview

- Question 1: Console application for book-author statistics.
- Question 2: Razor Pages `Book` module with `List` and `Update` pages.

## Question 1

Create a console application.

Requirements:
- Search books by title keyword.
- Filter by author.
- Sort by title or publish year.
- Count number of books by author.

## Question 2

Use `2/database.sql` and create Razor Pages.

Create page `/Book/List`:
- Search by title.
- Filter by author.
- Display update link for each book.

Create page `/Book/Update`:
- Accept `id` in query string.
- Update title, publish year and author.
