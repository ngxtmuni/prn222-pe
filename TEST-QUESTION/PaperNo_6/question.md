# PaperNo_6

## Overview

- Question 1: Console application for student-course statistics.
- Question 2: Razor Pages `Student` module with `List` and `Detail` pages.

## Question 1

Create a console application.

Requirements:
- Show all students.
- Filter students by course.
- Count number of students in each course.
- Sort by student name.
- Find the student who enrolls in the most courses.

## Question 2

Use `2/database.sql` and create Razor Pages.

Create page `/Student/List`:
- Display all students.
- Filter by course.
- Student name links to detail page.

Create page `/Student/Detail`:
- Accept `id` in query string.
- Display student information.
- Display all enrolled courses.
